#### jumpy-dot
https://github.com/mattaniak/jumpy-dot

#### What's that?
It's a simple ASCII-based game written in Python 3.
You're the player (green) and your mission is avoid enemies (reds).

#### Requiremets:
- *nix system,
- Python 3 installed.

#### Playing:
- ./main.py to run,
- 'Enter' key to jump.
