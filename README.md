### jumpy-dot
The license can be found in the root folder.

#### What's that?
It's a simple ASCII-based game written in Python 3.
Created without curses and any other similar modules.
Your mission (as a green) is jumping above enemies (reds).

#### Requiremets:
- *nix system,
- Python 3 installed.

#### Playing:
- ./main.py to run,
- 'Enter' key to jump.
